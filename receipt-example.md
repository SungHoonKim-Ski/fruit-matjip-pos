# 영수증 출력 예시 (58mm, 32자 폭)

## 일반배달 — 자동출력 (결제 직후, 배달예정시간 미정)

```
================================
                                  ← setTextQuadArea + bold
      과일맛집1995                ← 4배 면적 (가로2x 세로2x), 16자 폭 기준
                                  ← setTextNormal
================================
주문번호: #123
주문일시: 2026-02-11 14:30
                                  ← 배달예정 생략 (아직 미정)
--------------------------------
홍길동 / 010-1234-5678
--------------------------------
사과            2  5,000  10,000
바나나           1  3,000   3,000
--------------------------------
상품합계:            13,000원
배달비(1.2km):        2,900원
================================
                                  ← setTextQuadArea + bold
합계:    15,900원                 ← 4배 면적, 16자 폭 기준
                                  ← setTextNormal
================================
서울시 강남구 역삼동 123-45
2층 201호
================================
```

---

## 예약배달 — 자동출력 (배달예정시간 존재)

```
================================

      과일맛집1995                ← setTextQuadArea + bold

================================
주문번호: #456
주문일시: 2026-02-11 10:00
배달예정: 15:10                   ← 예약배달이면 항상 출력
--------------------------------
김철수 / 010-9876-5432
--------------------------------
딸기            1 15,000  15,000
망고            2  8,000  16,000
--------------------------------
상품합계:            31,000원
배달비(2.5km):        3,500원
================================

합계:    34,500원                 ← setTextQuadArea + bold

================================
서울시 서초구 반포동 67-8
3층 302호
================================
```

## 재출력 — 배달 목록에서 수동 출력 (배달예정시간 설정됨)

```
================================

      과일맛집1995                ← setTextQuadArea + bold

================================
주문번호: #123
주문일시: 2026-02-11 14:30
배달예정: 15:10                   ← 접수 후 설정된 시간 표시
--------------------------------
홍길동 / 010-1234-5678
--------------------------------
사과            2  5,000  10,000
바나나           1  3,000   3,000
--------------------------------
상품합계:            13,000원
배달비(1.2km):        2,900원
================================

합계:    15,900원                 ← setTextQuadArea + bold

================================
서울시 강남구 역삼동 123-45
2층 201호
================================
```

## ESC/POS 크기 명령 정리

| 메서드 | 가로 | 세로 | 폭 기준 | 적용 위치 |
|--------|------|------|---------|-----------|
| `setTextNormal()` | 1x | 1x | 32자 | 본문 전체 |
| `bold(true)` | 1x | 1x | 32자 | - |
| `setTextQuadArea()` | 2x | 2x | 16자 | 상호명, 합계 |
